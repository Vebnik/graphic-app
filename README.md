# How to launch

### Manual

- install `npm` and `node>=21`
- clone this repo `git clone this-repo`
- install deps `npm install`
- run dev `npm run dev`

### Docker

- clone this repo `git clone this-repo`
- run `docker compose -p graphic-app up --build -d`

### Notes

App availible on

```
ui:    http://localhost:80/
```
